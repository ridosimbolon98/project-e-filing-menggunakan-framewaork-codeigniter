HTML Heading

HTML Heading digunakan untuk mendeskripsikan heading (judul). HTML heading ditampilkan dengan ukuran huruf yang lebih besar dibanding teks biasa., dimulai dari h1 untuk ukuran teks terbesar sampai dengan h6 untuk ukuran huruf heading terkecil


HTML horizontal rule

Elemen ini diguakan untuk memberikan gais horinzontal lurus pada halaman web, didefinisikan dengan tag hr.


HTML paragraf

Elemen ini digunakan untuk mendeskripsikan sebuah paragraf, didefinisikan dengan pasangan tag p.


HTML line break

Elemen ini digunakan untuk membuat baris baru (line break), didefinisika dengan tag br. Dalam dokumen HTML spasi lebih maupun baris kosong akan dianggap sebagai spasi tunggal, sehingga untuk memberikan line break dapat digunakan elemen break line.
