HTML heading

HTML heading digunakan untuk mendeskripsikan heading (judul). HTML heading ditampilkan dengan ukuran huruf yang lebih besar dibanding teks biasa, dimulai dari h1 untuk ukuran teks terbesar sampai dengan h6 untuk ukuran teks terkecil.

HTML horizontal rule

Elemen ini digunakan untuk memberikan garis horizontal lurus pada halaman web, didefinisikan degan tag hr

HTML paragraf

Elemen ini digunakan untuk mendefinisikan sebuah paragraf, didefinisikan dengan pasangan tag p.

HTML line break

Elemen ini digunakan untuk membuat baris baru (line break), didefinsikan dengan tag br. Dalama dokumen HTML spasi lebih maupun baris kosong akan dianggap sebagai spasi tunggal, sehingga untuk memberikan line break dapat digunakan elemen break line.
